# Closetly - Assets Placeholder

Para ejecutar la aplicación, necesitarás agregar los siguientes assets en esta carpeta:

## 📱 Assets Requeridos

### icon.png (1024x1024px)
- Logo principal de la app
- Fondo transparente o de color primario (#62D9C7)
- Debe contener el ícono/logo de Closetly

### splash.png (1242x2436px)
- Pantalla de splash al iniciar
- Fondo color primario (#62D9C7)
- Logo centrado

### adaptive-icon.png (1024x1024px - Android)
- Versión del ícono para Android
- Área segura en el centro (512x512px)
- Fondo color primario

### favicon.png (48x48px - Web)
- Favicon para versión web
- Versión simplificada del logo

## 🎨 Generación Rápida de Assets

### Opción 1: Usar generadores online
- [App Icon Generator](https://appicon.co/)
- [Expo Asset Generator](https://www.appicon.co/#app-icon)

### Opción 2: Crear manualmente
1. Diseña un logo simple en Figma, Canva o Photoshop
2. Usa los colores de la paleta:
   - Primario: #62D9C7
   - Secundario: #6A4BFF
3. Exporta en las resoluciones requeridas

### Opción 3: Usar placeholders temporales
Por ahora, puedes usar colores sólidos o emojis:
- icon.png → 👔 sobre fondo #62D9C7
- splash.png → Texto "Closetly" sobre fondo #62D9C7

## 💡 Sugerencias de Diseño

El logo debería representar:
- Un gancho de ropa (coat hanger)
- Un closet/armario
- Prendas organizadas
- Minimalista y moderno

Colores recomendados:
- Usar el turquesa (#62D9C7) como color principal
- Detalles en púrpura (#6A4BFF)
- Tipografía moderna y limpia

---

**Nota**: Mientras no tengas assets listos, Expo usará placeholders por defecto.
